Overview

TagSelect is a free, web-based R-Shiny application designed for researchers who need to efficiently select random samples from participant lists and conduct experimental randomization. It also provides multiple file format support, and professional export capabilities.

Problems TagSelect Solves

📊
Large Dataset HandlingEfficiently processes up to 10,000 participants × 1,000 variables with optimized performance using R's sample.int() and Fisher-Yates shuffling.
🔄
Multiple Randomization MethodsSupports simple, complete, block, cluster, and stratified randomization for experimental designs via the randomizr package.
📁
Multi-Format SupportReads CSV, Excel (.xlsx/.xls), SPSS (.sav), Stata (.dta), and even Word (.docx) tables directly.
📤
Flexible Export OptionsDownloads full lists or selected participants in Excel, Word, or CSV formats with one click.

Features & Capabilities

TagSelect provides a complete random sampling and experimental randomization workflow within a single application.

RequirementVersionNotes

Local Installation

Step 1 — Download app.R

Download the source file from the GitHub repository: github.com/mudassiribrahim12/TagSelect

Step 2 — Install required packages

Run the following in your R console:

install.packages(c(
  "shiny", "readxl", "haven",
  "officer", "flextable", "writexl",
  "shinythemes", "dplyr", "DT",
  "shinyjs", "shinyWidgets", "randomizr",
  "tidyr", "purrr", "tools"
))

Step 3 — Load the libraries

library(shiny)
library(readxl)
library(haven)
library(officer)
library(flextable)
library(writexl)
library(shinythemes)
library(dplyr)
library(DT)
library(shinyjs)
library(shinyWidgets)
library(randomizr)
library(tidyr)
library(purrr)
library(tools)

Step 4 — Set working directory and run

Replace the path below with the location of your saved app.R file:

# Windows
setwd("C:/Users/YourName/Desktop/")

# Linux / macOS
setwd("/home/yourname/Desktop/")

# Launch the app
library(shiny)
runApp("app.R")

Step 5 — Open in browser

The app opens in the RStudio viewer pane. Click Open in Browser for the best experience.

Deployment

Deploy your own instance to shinyapps.io using the rsconnect package.

# Install rsconnect
install.packages("rsconnect")

# Deploy — replace with actual path to your app folder
rsconnect::deployApp("path/to/TagSelect")
⚠️ Note

Replace "path/to/TagSelect" with the actual folder path containing your app.R file.

Analysis Workflow

Follow these steps inside TagSelect to perform a complete random sampling or experimental randomization.

1
Navigate to "Simple Randomization" or "Experimental Randomization"

Choose the appropriate tab based on your research design needs.

2
Upload Your Dataset

Click Browse Files… and select your file. Supported formats: CSV, Excel (.xlsx/.xls), Stata (.dta), SPSS (.sav), Word (.docx).

3
Set Sample Size or Randomization Parameters

For simple randomization, enter the number of participants to sample. For experimental randomization, select the method and number of treatment groups.

4
Perform Randomization

Click Select Random Sample or Perform Randomization. The app uses R's efficient sampling algorithms.

5
Review Results

Selected participants are highlighted in the interactive data table. For experimental designs, the assignment column shows group allocation.

6
Download Results

Click download buttons for Excel, Word, or CSV formats. Choose between full list or only selected participants.

Randomization Methods

The Experimental Randomization tab provides five distinct randomization methods for research designs.

MethodDescriptionUse Case
Simple RandomizationEach participant independently assigned with equal probability to treatment groups.Basic designs with no blocking or stratification.
Complete RandomizationFixed number of participants assigned to each group, randomly permuted.When exact group sizes are required.
Block RandomizationParticipants grouped into blocks, then randomly assigned within each block.Ensuring balance over time or across sites.
Cluster RandomizationGroups (clusters) of participants are randomized together.When intervention is delivered at group level (e.g., schools, clinics).
Stratified RandomizationRandomization performed separately within each stratum defined by a categorical variable.Ensuring balance across key prognostic factors (e.g., gender, age group).
📐 Technical Implementation

TagSelect uses the randomizr package for experimental randomization: simple_ra(), complete_ra(), block_ra(), cluster_ra(), and strata_ra(). Simple randomization uses R's sample.int() with Fisher-Yates shuffling for efficiency.

Exporting Results

TagSelect provides multiple export options accessible from the sidebar of each tab.

Export TypeFormatContentsNotes
Download Full ListExcel / Word / CSVComplete dataset with Selected column indicating sampled participants.Word exports use flextable with zebra striping.
Download Selected OnlyExcel / Word / CSVOnly participants marked as Selected (sampled).Useful for extracting the random sample.
Download Randomized DataCSVExperimental randomization results with Assignment column.Available only in Experimental Randomization tab.
⚠️ Before Downloading

You must perform randomization (click "Select Random Sample" or "Perform Randomization") at least once before downloading. Attempting to download without running the analysis will result in an error.

Assumptions

🎲
Random Sampling

Simple randomization assumes each participant has an equal and independent chance of selection.

📊
Complete Data

Missing values are automatically removed via na.omit() before randomization.

🔗
Independent Observations

Randomization methods assume observations are independent unless using cluster randomization.

🔢
Adequate Sample Size

Experimental designs require sufficient participants for the chosen randomization method (e.g., block size must divide total N).

Limitations

📉
No Covariate Adjustment

Stratified randomization uses the specified grouping variable but does not adjust for multiple covariates.

🧮
No Post-Allocation Analysis

The app performs randomization only; statistical analysis of outcomes is not included in this version.

🌐
Session-Based Only

Results are not saved between sessions. Download your results before closing the browser.
